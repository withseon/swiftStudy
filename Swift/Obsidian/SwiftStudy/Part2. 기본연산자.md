
## 기본 연산자
1. 단항 연산자 / 이항 연산자 / 삼항 연산자
2. 할당 연산자( = )
3. 산술 연산자 (+, - \*, /, %)

## 복합할당 / 비교 / 논리 연산자 / 연산의 우선순위 / 접근 연산자
1. 복합 할당 연산자 ( +=, -=, \*=, /=, %=) : -한 다음에 다시 할당한다.
2. 비교 연산자 (\==, !=, <. <=, >, >=)
3. 논리 연산자(!, &&, ||)
4. 연산의 우선순위: 괄호 > 단항 > 산술(\* / %) > 산술(+, -) > 비교 > 논리 > 삼항 > 할당
5. 접근 연산자(.) : 하위 개념으로 접근

### 패턴 매칭 연산자( ~= )
	표현식의 범위에 매칭이 되면 true, 아니면 false
> 0...50 ~= 30 의 결과값은 `true
> 51...100 ~= 30 의 결과값은 `false
